# MotifLab for Suno

這是一個 Chrome / Edge 可載入的 Manifest V3 插件。

## 安裝

1. 打開 Chrome 或 Edge 的擴充功能頁面。
2. 開啟「開發人員模式」。
3. 點「載入未封裝項目」。
4. 選擇這個資料夾：`suno-extension`。
5. 打開 `https://suno.com/`，右下角會出現 `M` 按鈕。

## 更新

如果已經載入過舊版，改完文件後需要到擴充功能頁面點一次「重新載入」，再刷新 Suno 分頁。只刷新 Suno 不一定會換到新的 content script。

## 用法

- 在 MotifLab 網頁版的提示詞系統中複製插件資料。
- 到 Suno 頁面的插件面板貼上並匯入。
- 在 Suno 使用期間可以隨時查看、編輯、複製或填入當前輸入框。
